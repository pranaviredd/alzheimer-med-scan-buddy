# Alzheimer's Stage Detection and Medicine Recommendation System

AI-powered healthcare application for detecting Alzheimer's disease stages from MRI brain scans and providing stage-specific medicine recommendations.

## Features

- MRI Image Classification
- CNN-Based Alzheimer's Detection
- Stage Prediction
- Medicine Recommendation Engine
- Patient Monitoring
- FastAPI Backend

## Architecture

```text
MRI Image
    ↓
Image Preprocessing
    ↓
CNN Model
    ↓
Stage Prediction
    ↓
Recommendation Engine
    ↓
Medicine Recommendation
    ↓
Patient Dashboard
```
